# Nevajen Capital — Static Site

## Deploy to GitHub Pages
1. Create a repo (e.g., `nevajen/nevajen.github.io`).
2. Upload the contents of this folder to the repo root.
3. In Settings → Pages, ensure the site is served from the **root** of the default branch.
4. Visit `https://nevajen.github.io/`.

## Files
- `index.html`, `services.html`, `faq.html`, `about.html`, `contact.html`
- `img/favicon.ico`, `img/favicon-32.png`, `img/og-image.png`
- `robots.txt`, `sitemap.xml`, `404.html`
- `README.md`
